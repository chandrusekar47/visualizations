Analysis of Global Terrorism Dataset 1970-2015

The goal of this project is to make it easier to identify where terrorism might occur in the future, by what means and organization it might occur, so that it can be prevented. This is an important problem today because of the already large and growing threat of terror by ISIS, Al-Qaeda, lone wolf attacks, and other organizations. This website provides several visualizations that can be used to identify different trends and patterns in terrorist attacks across the world.

Interesting Patterns

The following patterns can be seen in the heatmap based visualization:
  • Increase in lone wolf attacks all over the USA in the last three years. [1] [2] [3]
  • Suspiciously low terrorist attacks in Iran and Saudi Arabia, even with all neighboring countries being involved in deep conflict. [4]
  • Growth and expansion of ISIL/ISIS starting in 2013.[5]
  • Resurgence of the Irish Republican Army in UK and Ireland in recent years.

Software used

  • R for data analysis and pre-processing
  • D3 for drawing barcharts, line graphs and scatter plots
  • Leaflet js for rendering the map layer and overlaying the country boundaries on top
  • heatmap.js to generate the heat map
